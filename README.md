> 🤖 本仓库通过 GitHub Actions 自动更新 GitHub 个人数据。
>
> - **自动更新**：每天 UTC 00:17（北京时间 08:17）自动执行
> - **手动触发**：在 [Actions](https://github.com/jincaiw/mygithub/actions) 页面点击 **Update Profile README** → **Run workflow**
> - **更新范围**：仅修改 `<!-- AUTO-GITHUB:START -->` 与 `<!-- AUTO-GITHUB:END -->` 之间的内容
> - **权限配置**：如推送失败，请在仓库 Settings → Actions → General → Workflow permissions 中勾选 **Read and write permissions**
