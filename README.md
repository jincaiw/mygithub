> ### ⚙️ 工作流说明
>
> | 步骤 | 操作 |
> |:---|---|
> | ① 检出代码 | `actions/checkout@v4` 拉取仓库最新代码 |

> | ④ 更新 README | 仅替换标记区域内的内容，其他部分不受影响 |
> | ⑤ 提交推送 | 有变化时自动提交，无变化跳过，不会产生空提交 |

>
> - **定时触发**：每周一 UTC 00:17（北京时间 08:17）自动执行
> - **手动触发**：[Actions 页面](https://github.com/jincaiw/jincaiw/actions) → **Update Profile README** → **Run workflow**
>
